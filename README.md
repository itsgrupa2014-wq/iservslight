
# Lineārais gaismeklis — WebGL konfigurators (bāze)

Statiskā demo lapa (HTML/CSS/JS + Three.js), kas ļauj:
- izvēlēties **garumu** (600/1200/1800/2400/3000 mm),
- **CCT** (3000K/4000K),
- **vadību** (DALI — pagaidām bez piemaksas),
- **montāžu** (ar/bez trosītēm),
- **RAL** krāsu (mini palete + manuāla ievade),
- redzēt **cenu (bez PVN)** un **jaudu** reāllaikā,
- lejupielādēt **specifikācijas failu** (demonstrācijas `.txt` — nākamajā versijā PDF).
